## 浏览器报告的错误

`finally{}`

`Error`
`EvalError` 错误使用eval函数
`RangeError` 数值超出范围
`ReferenceError` 使用未生命变量
`SyntaxError` 语法错误
`TypeError` 变量类型不符合要求
`URIError` URI格式不对

`throw`抛出错误
没有通过`try-catch`处理的错误触发window对象的`error`事件
